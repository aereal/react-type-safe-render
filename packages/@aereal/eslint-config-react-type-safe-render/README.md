# eslint-config-react-type-safe-render

## Installation

```
npm i -D @aereal/eslint-config-react-type-safe-render
```

```
yarn add -D @aereal/eslint-config-react-type-safe-render
```

## Usage

```json
{
  "extends": [
    "@aereal/eslint-config-react-type-safe-render"
  ]
}
```
